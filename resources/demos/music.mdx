This demo showcases Meilisearch's ability to handle massive datasets with instant search across 40 million songs from the MusicBrainz database.

## Features demonstrated

- **Large-scale search**: 40 million documents indexed
- **Instant results**: Sub-100ms response times
- **Rich metadata**: Artist, album, year, and duration
- **Typo tolerance**: Find songs even with misspelled names

## Use case

Music databases present unique search challenges:

- Artists and song titles from many languages
- Common misspellings of artist names
- Users search by partial song titles
- Need for instant feedback while typing

## Technical highlights

| Feature | Value |
|---------|-------|
| Dataset size | 40 million songs |
| Source | MusicBrainz database |
| Searchable fields | Title, artist, album |
| Average response | Under 100ms |

## Performance at scale

This demo proves Meilisearch handles large datasets efficiently:

- Indexing 40M documents completes in reasonable time
- Search latency remains consistent regardless of dataset size
- Memory-efficient storage using LMDB
- No degradation as index grows
