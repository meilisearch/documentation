Search through the Museum of Modern Art's extensive collection of artworks. This demo showcases how Meilisearch can power cultural institution search experiences.

## Features demonstrated

- **Art catalog search**: Search by artist, title, medium, or date
- **Rich metadata display**: Artwork details, dimensions, acquisition info
- **Faceted filtering**: Filter by artist, nationality, medium, department
- **Cultural data handling**: Multi-language artist names and titles

## Use case

Museum and cultural institution collections have specific needs:

- Artworks have extensive metadata (artist, date, medium, dimensions)
- Multiple ways to discover content (by artist, period, style)
- Need to handle historical and international names
- Visual presentation of results with artwork images

## Dataset

The MoMA collection includes:

- Paintings, sculptures, and installations
- Photography and prints
- Design objects and architecture
- Film and media

## Technical highlights

| Feature | Implementation |
|---------|----------------|
| Searchable fields | Title, artist, medium, classification |
| Filterable fields | Department, nationality, date range |
| Display attributes | Image, dimensions, credit line |
| Special handling | Date parsing for historical works |
