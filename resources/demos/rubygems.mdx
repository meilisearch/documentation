Search through the complete RubyGems package registry. This demo showcases how Meilisearch can power package discovery for developer tools and registries.

## Features demonstrated

- **Package search**: Find Ruby gems by name or description
- **Version info**: See latest versions and download counts
- **Relevancy tuning**: Popular packages ranked higher
- **Instant results**: Fast search across thousands of packages

## Use case

Package registries benefit from smart search:

- Developers search by partial or misspelled names
- Popular packages should rank higher
- Descriptions help discover new packages
- Download counts indicate reliability

## Technical highlights

| Feature | Implementation |
|---------|----------------|
| Dataset | Full RubyGems registry |
| Searchable | Name, description, authors |
| Ranking boost | Download count as custom ranking |
| Display | Version, downloads, dependencies |

## Custom ranking example

```json
{
  "rankingRules": [
    "words",
    "typo",
    "proximity",
    "attribute",
    "sort",
    "exactness",
    "downloads:desc"
  ]
}
```

This configuration ranks results by relevance first, then by download count for equally relevant results.
