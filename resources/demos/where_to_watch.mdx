Where to Watch is a movie discovery application that helps users find which streaming platforms offer the movies they want to watch. Built with Meilisearch, it demonstrates how to create a fast, responsive search experience across a large movie database.

[![Movies demo showing search results](/assets/images/movies-demo-dark.gif)](https://where2watch.meilisearch.com)

## Features demonstrated

- **Instant search**: Results appear as you type with sub-50ms response times
- **Typo tolerance**: Find movies even with misspelled titles
- **Faceted filtering**: Filter by genre, release year, and streaming platform
- **Rich result display**: Movie posters, descriptions, ratings, and platform availability

## Use case

This demo showcases a common search pattern for media catalogs:

- Users search for movies by title or keywords
- Results include rich metadata (posters, ratings, descriptions)
- Filters help narrow down by preferences
- Platform availability helps users decide where to watch
