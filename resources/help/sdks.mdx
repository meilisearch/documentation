## Quick installation

<Tabs>
  <Tab title="JavaScript">
```bash
npm install meilisearch
# or
yarn add meilisearch
```

```javascript
import { MeiliSearch } from 'meilisearch'

const client = new MeiliSearch({
  host: process.env.MEILISEARCH_URL,
  apiKey: process.env.MEILISEARCH_API_KEY
})

const results = await client.index('movies').search('star wars')
```
  </Tab>
  <Tab title="Python">
```bash
pip install meilisearch
```

```python
import meilisearch
import os

client = meilisearch.Client(
    os.environ.get('MEILISEARCH_URL'),
    os.environ.get('MEILISEARCH_API_KEY')
)

results = client.index('movies').search('star wars')
```
  </Tab>
  <Tab title="PHP">
```bash
composer require meilisearch/meilisearch-php
```

```php
use MeiliSearch\Client;

$client = new Client(
    getenv('MEILISEARCH_URL'),
    getenv('MEILISEARCH_API_KEY')
);

$results = $client->index('movies')->search('star wars');
```
  </Tab>
  <Tab title="Ruby">
```bash
gem install meilisearch
```

```ruby
require 'meilisearch'

client = MeiliSearch::Client.new(
  ENV['MEILISEARCH_URL'],
  ENV['MEILISEARCH_API_KEY']
)

results = client.index('movies').search('star wars')
```
  </Tab>
  <Tab title="Go">
```bash
go get github.com/meilisearch/meilisearch-go
```

```go
import (
    "os"
    "github.com/meilisearch/meilisearch-go"
)

client := meilisearch.New(
    os.Getenv("MEILISEARCH_URL"),
    meilisearch.WithAPIKey(os.Getenv("MEILISEARCH_API_KEY")),
)

results, _ := client.Index("movies").Search("star wars", nil)
```
  </Tab>
</Tabs>

## Framework integrations

### Laravel

Laravel Scout has built-in Meilisearch support:

```bash
composer require laravel/scout
php artisan vendor:publish --provider="Laravel\Scout\ScoutServiceProvider"
```

```php
// .env
SCOUT_DRIVER=meilisearch
MEILISEARCH_HOST=your-meilisearch-url
MEILISEARCH_KEY=your-api-key
```

[Laravel Scout guide →](/guides/laravel_scout)

### Ruby on Rails

```bash
gem install meilisearch-rails
```

```ruby
class Article < ApplicationRecord
  include MeiliSearch::Rails

  meilisearch do
    attribute :title, :content
    searchable_attributes [:title, :content]
  end
end
```

[Rails guide →](/guides/ruby_on_rails_quick_start)

### Symfony

```bash
composer require meilisearch/search-bundle
```

[Symfony integration →](https://github.com/meilisearch/meilisearch-symfony)

### Strapi

```bash
npm install strapi-plugin-meilisearch
```

[Strapi guide →](/guides/strapi_v4)

### Quick InstantSearch example

```html
<script src="https://cdn.jsdelivr.net/npm/@meilisearch/instant-meilisearch/dist/instant-meilisearch.umd.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/instantsearch.js@4"></script>

<script>
const MEILISEARCH_URL = "https://your-instance.meilisearch.io";
const MEILISEARCH_API_KEY = "your-search-key";

const search = instantsearch({
  indexName: "movies",
  searchClient: instantMeiliSearch(MEILISEARCH_URL, MEILISEARCH_API_KEY).searchClient
});

search.addWidgets([
  instantsearch.widgets.searchBox({ container: "#searchbox" }),
  instantsearch.widgets.hits({ container: "#hits" })
]);

search.start();
</script>
```
