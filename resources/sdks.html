# SDKs and Integrations

You can use MeiliSearch API wrappers in your favorite language. These libraries support all API routes.

  • Golang
  • JavaScript
  • Laravel Scout
  • PHP
  • Python
  • Ruby
  • Rust
  • Swift

# Front-End Tools

  • VuePress plugin
  • docs-searchbar.js: a search bar integration for all kinds of documentation.

# Other Tools

  • docs-scraper: a scraper tool to automatically read the content of your documentation and store it into MeiliSearch.
  • meilisearch-digital-ocean: how to deploy a MeiliSearch instance on DigitalOcean. See this dedicated page to use it.
