A project is an isolated Meilisearch instance. Follow these steps to create one from the Meilisearch Cloud dashboard.

## Prerequisites

- A Meilisearch Cloud account. [Sign up at cloud.meilisearch.com](https://cloud.meilisearch.com) if you do not have one.

## Steps

### 1. Go to the dashboard

Open [cloud.meilisearch.com](https://cloud.meilisearch.com) and sign in. You will land on the projects list.

### 2. Click "New project"

Click the **New project** button in the top-right corner of the projects list.

### 3. Fill in the project form

Enter the following details:

| Field | Description |
|-------|-------------|
| **Name** | A display name for your project. You can rename it later. |
| **Region** | The cloud region where your project will be hosted. Choose the region closest to your users. See [Regions](/capabilities/platform/infrastructure/regions) for guidance. |
| **Plan** | The resource tier (CPU, RAM, storage). You can change this later. See [Manage resources](/capabilities/platform/infrastructure/manage_resources) for details. |

### 4. Confirm and create

Click **Create project**. Meilisearch Cloud provisions the instance. This typically takes a few seconds.

### 5. Access your project

Once ready, you are redirected to the project dashboard. From there you can:

- Copy your **API keys** to start indexing documents
- Open the **search preview** to test queries
- Configure **settings** such as filterable attributes and ranking rules
