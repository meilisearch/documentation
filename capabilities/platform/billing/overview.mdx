Meilisearch Cloud billing is straightforward: you are charged per project based on the resource tier and region you choose. There are no per-seat fees or query limits.

## Billing basics

| Concept | Details |
|---------|---------|
| **Billing unit** | Per project |
| **Billing cycle** | Daily (prorated if project created or deleted mid-day) |
| **Price factors** | Resource tier (CPU/RAM/storage) and region |
| **Payment processor** | Stripe |
| **Invoices** | Generated automatically each billing period |

## What affects your bill

Only active projects generate charges. Deleting a project stops billing for that project immediately (prorated for the current day). Creating a project immediately starts billing.

Team members and API keys do not affect billing.
