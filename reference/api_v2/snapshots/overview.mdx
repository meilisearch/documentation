The `/snapshots` route allows you to create fast, version-specific backups of your Meilisearch database.

<Note>
Snapshots are faster to create than dumps but can only be used with the same Meilisearch version.
</Note>

## Dumps vs Snapshots

| Feature | Dumps | Snapshots |
|:--------|:------|:----------|
| **Portability** | Any version | Same version only |
| **Speed** | Slower | Faster |
| **Format** | Raw data | Binary database |
| **Use case** | Migration | Quick backup |
