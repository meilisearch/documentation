The `/dumps` route allows you to create portable backups of your Meilisearch database.

<Tip>
Dumps are ideal for migrating between Meilisearch versions or moving data between servers. They can be imported on any Meilisearch instance.
</Tip>

## Dumps vs Snapshots

| Feature | Dumps | Snapshots |
|:--------|:------|:----------|
| **Portability** | Any version | Same version only |
| **Speed** | Slower | Faster |
| **Format** | Raw data | Binary database |
| **Use case** | Migration | Quick backup |
