---
openapi: post /indexes/{indexUid}/documents/delete
---

Delete a selection of documents based on array of document IDs or a filter.

<Warning>
Filtering is not possible without first explicitly adding the attributes you want to filter on to the [`filterableAttributes` list](/reference/api/settings/filterable-attributes#update-filterable-attributes).
</Warning>
