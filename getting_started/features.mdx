Meilisearch provides a comprehensive set of search features designed to deliver fast, relevant results for any use case.

## Search capabilities

### Full-text search

Lightning-fast keyword search with typo tolerance and customizable relevancy.

| Feature | Description |
|---------|-------------|
| [Typo tolerance](/products/full_text_search/typo_tolerance_settings) | Get relevant results even with spelling mistakes |
| [Ranking rules](/products/full_text_search/ranking_rules) | Customize how results are ordered |
| [Synonyms](/products/full_text_search/synonyms) | Define equivalent terms for better recall |
| [Stop words](/reference/api/settings/stop-words) | Ignore common words like "the" or "and" |

[Learn more about full-text search →](/products/full_text_search/overview)

### AI-powered search

Semantic and hybrid search using vector embeddings for meaning-based results.

| Feature | Description |
|---------|-------------|
| [Hybrid search](/products/ai_powered_search/getting_started) | Combine keyword and semantic search |
| [Embedders](/products/ai_powered_search/choose_an_embedder) | OpenAI, Hugging Face, Cohere, and more |
| [Similar documents](/products/ai_powered_search/retrieve_related_search_results) | Find related content automatically |
| [Image search](/products/ai_powered_search/image_search_with_multimodal_embeddings) | Search images with multimodal embeddings |

[Learn more about AI-powered search →](/products/ai_powered_search/overview)

## Language support

Meilisearch provides optimized support for many languages:

- **Latin-based languages**: English, French, Spanish, German, Italian, Portuguese, etc.
- **CJK**: Chinese, Japanese, Korean with specialized tokenization
- **RTL languages**: Hebrew, Arabic
- **Others**: Thai, Greek, and more
