## Meilisearch Cloud vs Self-hosting

| | Cloud | Self-hosted |
|---|---|---|
| **Setup time** | Minutes | Hours to days |
| **Maintenance** | Managed | You handle it |
| **Scaling** | Automatic | Manual |
| **Updates** | Automatic | Manual |
| **Best for** | Most users | Full control needs |

[Meilisearch Cloud](https://www.meilisearch.com/cloud?utm_campaign=oss&utm_source=docs&utm_medium=overview) is the recommended way to use Meilisearch. Start with a [14-day free trial](https://cloud.meilisearch.com).
