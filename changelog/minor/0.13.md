# v0.13

First released: 2020-08-03

## New Features

### Placeholder Search
You can now use placeholder search functionality, allowing for more flexible search queries.

### Wildcard Support for Attributes
Displayed and searchable attributes now support wildcard characters, enhancing your search capabilities.

### Database Version Mismatch Check
A new check for database version mismatches has been added, ensuring compatibility and preventing potential issues.

## Improvements

### Enhanced Error Codes in Updates
Error codes have been improved in update responses, providing clearer information about issues that may arise during operations.

### Sentry Disable Argument
A new argument has been introduced to disable Sentry logging, giving you more control over your error tracking setup.

### Log Analytics Feature
If enabled, analytics logging will now be performed, allowing for better tracking of usage and performance metrics.

## Other

### Removal of sys-info Route
The `sys-info` route has been removed. Please update your integrations accordingly.
