# v0.10

First released: 2020-04-11

## New Features

### Floating Point Support in Filters
You can now use floating point numbers in your filters, allowing for more precise queries.

### Tokenizer Separator Enhancement
The `@` character has been added as a tokenizer separator, improving how text is processed during searches.

### Array of Strings Filtering
Filtering on arrays of strings is now supported, enabling more complex queries on your data.

### Number of Hits in Search Results
Search results now include the total number of hits, providing better insight into the results returned.

### Aligned Crop Support in Search Results
You can now use aligned crop functionality in your search results, enhancing how images are displayed.

### Support for Nested Values
The system now supports nested `null`, `boolean`, and sequence values, allowing for more complex data structures.

### Wildcard Support
Wildcard functionality has been added for attributes to retrieve, highlight, and crop, giving you more flexibility in your queries.

## Improvements

### Refined Filtering
Filtering has been improved for better performance and accuracy in search results.

### Sanitized Web Interface Content
The content displayed in the web interface has been sanitized for a cleaner and safer user experience.

### Core Benchmark Fix
The core benchmark has been fixed to ensure accurate performance measurements.

### CORS Issues Resolved
CORS issues have been addressed, improving cross-origin requests.

## Other

### ARMv7 and ARMv8 Binaries
ARMv7 and ARMv8 binaries are now published in the releases, expanding compatibility with different architectures.
