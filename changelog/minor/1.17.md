# v1.17

First released: 2025-08-12

## New Features

### Chat Completions Route
You can now enhance your search experience with the new chat completions route. This feature allows you to turn your queries into conversational interactions, making your search feel more natural. It is easy to set up and works seamlessly with your favorite language models.

## Improvements

### Webhook API Support
The Webhook API is now supported, allowing you to receive real-time notifications about events in your Meilisearch instance. This can help you integrate with other services and automate workflows.

### Optimized `STARTS_WITH` Filter
The performance of the `STARTS_WITH` filter has been optimized and stabilized. You no longer need to activate the experimental feature to use this operator, making it easier to implement in your searches.

### Docker Image Signing
The Docker image is now signed using Cosign in keyless mode, enhancing the security of your deployments.

## Other

### Breaking Change: Chat Settings Update
The chat settings endpoint has changed from `PUT` to `PATCH`. This means that any integrations or SDKs that rely on the previous method will need to be updated to accommodate this change.
