# v1.21

First released: 2025-09-15

## New Features

### New Vector Store Backend
A new vector store backend has been introduced to enhance performance, particularly when using binary quantization. To enable this feature, set the `vectorStoreSetting` to experimental and update the `vectorSetting` for the desired indexes:

```json
{
  "vectorStoreSetting": "experimental",
  "vectorSetting": "experimental"
}
```

### Persian Language Support
Support for the Persian language has been added, allowing for better handling of Persian text in your documents.

## Improvements

### Enhanced Performance with Vector Store
The new vector store backend provides improved performance, making it more efficient for applications that utilize vector searches.
