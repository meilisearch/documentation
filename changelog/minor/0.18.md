# v0.18

First released: 2021-01-11

## New Features

### New Tokenizer Integration
The latest version introduces integration with a new tokenizer, enhancing text processing capabilities and improving search accuracy.

## Improvements

### Increased Default Payload Size
The default payload size has been increased, allowing for larger data submissions in requests, which can improve performance when handling extensive datasets.

### Code Quality Enhancements
Various improvements have been made to the code quality, resulting in a more stable and maintainable codebase.

## Other

### Setting Consistency Bug Fix
A bug related to setting consistency has been addressed, ensuring more reliable behavior when configuring settings.

### Attributes to Retrieve Bug Fix
A fix has been implemented for a bug affecting the retrieval of specific attributes, enhancing the accuracy of search results.
