# v1.29

First released: 2025-12-08

## New Features

### Enhanced Settings Indexer
The new settings indexer now supports `searchableAttributes` and `exactAttributes`, allowing for better scalability and near-instant cancellations during indexing operations. This change means that any combination of the following settings will enable the new indexer:
- `searchableAttributes`
- `exactAttributes`
- `proximityPrecision`
- `embedders`

For OSS users who wish to disable this new indexer, set the `MEILI_EXPERIMENTAL_NO_EDITION_2024_FOR_SETTINGS` environment variable to `true`.

### Default Vector Store for New Indexes
From version 1.29.0, newly created indexes will use the new vector store backend by default, which offers improved performance and relevancy. Existing indexes will remain unchanged, but you can still choose the vector store backend explicitly.

### Support for XLM Roberta Models
The `huggingFace` embedder now supports models with the XLM Roberta architecture, allowing for local generation of embeddings on both CPU and GPU.

## Improvements

### Build Compatibility Note
The `git` binary is now required at build time to populate the `commitSha1` field of the `/version` endpoint. This change was implemented to enhance build performance.

## Other

### Asset Availability Note
The Meilisearch binary is currently unavailable for `meilisearch-enterprise-macos-amd64` and `meilisearch-macos-amd64` due to a compilation oversight. However, the binaries for `meilisearch-enterprise-macos-apple-silicon` and `meilisearch-macos-apple-silicon` are available.
