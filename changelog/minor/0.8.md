# v0.8

First released: 2019-11-26

## New Features

### New Default Port
MeiliSearch now exposes the port `7700` instead of `8080`. This change helps avoid conflicts with common existing TCP servers, such as Node.js.

## Improvements

### Updated Output Format for Updates
The output format for updates has been changed to provide clearer information. This enhancement improves the readability and usability of the update responses.

### Removal of `libssl` Runtime Dependency
The `libssl` runtime dependency has been removed, simplifying the installation process for users. This change makes it easier to set up MeiliSearch without dealing with complex dependencies.

### Asynchronous Update Processing
You can now send updates to the HTTP server without waiting for it to process anything other than the body parsing. This improvement enhances the efficiency of your interactions with the server.

## Other

### Breaking Changes in HTTP APIs
This release introduces significant changes to many HTTP APIs. Users are encouraged to refer to the updated documentation for guidance on these changes.
