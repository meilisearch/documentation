# v1.19

First released: 2025-08-25

## New Features

### Automatic Document Sharding
Meilisearch now supports automatic sharding of documents across multiple instances. This feature allows for better scalability and performance in handling large datasets.

For implementation details, refer to the documentation on sharding.

## Improvements

### Enhanced Hybrid Search Performance
The performance of hybrid searches combined with filters has been significantly improved. Previously, using filters could drastically increase search times. Now, Meilisearch directly computes the semantic distance with filtered candidates when only a few candidates are returned from the filter, optimizing the search process.

Example of a hybrid search with a filter:

```json
{
  "q": "hello world",
  "limit": 100,
  "filter": "tag=science",
  "hybrid": {
    "semanticRatio": 0.5,
    "embedder": "default"
  }
}
```
