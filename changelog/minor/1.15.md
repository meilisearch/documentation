# v1.15

First released: 2025-06-09

## New Features

### Experimental Chat Feature
You can now enable an experimental chat feature that allows you to create a chat workspace with your indexes. This feature enables LLM-driven chat capabilities.

To set up the chat workspace, use the following command:

```bash
curl -X POST 'http://localhost:7700/chats/my-assistant/settings' \
 -H 'Content-Type: application/json' \
 -d '{
 "source": "openAi",
 "apiKey": "sk-abc..."
 }'
```

Once set up, you can interact with your indexes using the OpenAI SDK:

```javascript
import OpenAI from 'openai';

const client = new OpenAI({
 baseURL: 'http://localhost:7700/chats/my-assistant',
 apiKey: 'YOUR_MEILISEARCH_CHAT_API_KEY',
});

const completion = await client.chat.completions.create({
 model: 'gpt-3.5-turbo',
 messages: [{ role: 'user', content: 'What is Meilisearch?' }],
 stream: true,
});

for await (const chunk of completion) {
 console.log(chunk.choices[0]?.delta?.content || '');
}
```

### Disable Typo Tolerance for Numbers
You can now disable typo tolerance specifically for numbers by setting `typoTolerance.disableOnNumbers` to `true`. This can help reduce false positives in search results.

To configure this setting, use the following command:

```bash
curl -X POST 'http://localhost:7700/indexes/movies/settings' \
 -H 'Content-Type: application/json' \
 -d '{
 "typoTolerance": {"disableOnNumbers": true}
 }'
```

### Lexicographic String Filters
This release introduces the ability to filter strings lexicographically using comparison operators such as `<`, `<=`, `>`, `>=`, and `TO`. This is particularly useful for filtering human-readable dates.

Example of using the new filter:

```bash
curl -X POST 'http://localhost:7700/indexes/movies/search' \
 -H 'Content-Type: application/json' \
 -d '{
 "filter": "release_date >= '2024-06'"
 }'
```

## Improvements

### Searchable Attributes Database Bug Fix
A major fix has been implemented to address an issue where some searchable fields were incorrectly removed from the searchable databases when they were removed from the `filterableAttributes` setting. This ensures that fields remain searchable as intended.

### Chat Route and Error Handling Fixes
The chat route now includes the base URL, and error handling for Mistral has been improved, enhancing the overall user experience.

### Embedding Regeneration Fixes
Various improvements have been made to the embedding regeneration process, ensuring better performance and reliability.

## Other

### Upgrade Rollback Capability
You can now cancel an upgrade to a new Meilisearch version by rolling back all upgraded indexes, providing more control during the upgrade process.

### SSL Certificate Support
Support for EC private keys as SSL certificates has been added, enhancing security options for users.

### CLI Option for Snapshot Compaction
A new CLI option allows users to stop compacting snapshots, which can speed up their generation.
