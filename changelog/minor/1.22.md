# v1.22

First released: 2025-09-24

## New Features

### Configurable Remote Search Timeout
The remote federated search timeout can now be configured. Set the environment variable `MEILI_EXPERIMENTAL_REMOTE_SEARCH_TIMEOUT_SECONDS` to a positive integer value to adjust the timeout duration for waiting nodes.

## Improvements

### Geo Backend for GeoJSON
A new geo backend has been introduced to store GeoJSON data and filter on polygons. You can now:
1. Make the `_geojson` field filterable.
2. Send your documents with a `_geojson` field filled with valid GeoJSON.
3. Filter your documents using the new `_geoPolygon` filter, along with the existing `_geoBoundingBox` and `_geoPoints` filters.

## Other

### Migration Note for Remote Federated Search Users
If you are not a remote federated search user, it is not necessary to migrate from v1.22.x.
