# v1.27

First released: 2025-11-24

## Improvements

### Enhanced Task Queue Error Reporting
Error messages in the task queue have been improved to provide better debugging information when uploading snapshots to S3. This enhancement will help users quickly identify and resolve issues during the upload process.

### Optimized Batch Size for Task Ingestion
The default total size for batched tasks is now set to half of the maximum indexing memory. This change improves the performance of task ingestion, allowing for more efficient processing of documents.

## Other

### Compatibility Support for macOS
Meilisearch binaries for macOS are now generated with support for macOS Sonoma (`macos-14`). This ensures compatibility with the latest macOS version for users on that platform.
