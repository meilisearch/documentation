# v0.23

First released: 2021-10-11

## New Features

### Geosearch Support
You can now filter and sort search results based on geographic criteria. This feature allows you to refine the precision, distance, and shape of the geographical areas in which documents should be searched.

### Support for New Dataset Formats
MeiliSearch now supports importing datasets in CSV and NDJSON formats, expanding the options for data ingestion.

## Improvements

### Docker Image Enhancement
The `curl` command-line tool has been added to the Docker image, making it easier to interact with the MeiliSearch API directly from the container.

### Memory Management Optimization
The memory allocator has been updated to use `tikv-jemallocator` instead of the previous `jemallocator`, which may improve performance and memory usage.

### CI Dependency Caching
Improvements have been made to the dependency cache for continuous integration, which should enhance build times.

### Unified Dependency Management
The pest dependency has been unified, streamlining the dependency management process.

### Documentation Improvements
The `CONTRIBUTING.md` file has been enhanced to provide clearer guidelines for contributors.

## Other

### Breaking Change: Re-indexation Required
This release requires re-indexation of all documents. Re-indexation will occur automatically when migrating your data from a previous version. 

### Mandatory Content-Type Header
It is now mandatory to set `Content-Type` headers when using API routes that accept a payload. This change ensures consistent behavior across all relevant routes. If you are using one of the integrations, this change will not impact your code.
