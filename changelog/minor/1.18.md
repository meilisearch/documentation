# v1.18

First released: 2025-08-18

## New Features

### Return `queryVector` in Search Response
When using the `retrieveVectors` parameter, the search response will now include the `queryVector`. This allows you to see the vector representation of the query used for the search.

### Retrieve Documents with Specific Vectors
You can now retrieve documents that include vectors from specific embedders. This feature enhances the flexibility of your search capabilities by allowing you to specify which vector embeddings to use.

### Rename Indexes via API
You can now rename indexes using the API. This feature simplifies the management of your indexes, allowing for better organization and clarity in your data structure.
