# v0.14

First released: 2020-08-31

## New Features

### Snapshotting
You can now create snapshots of your indexes, allowing you to save the current state of your data for backup or recovery purposes.

## Improvements

### Improved Facet Distribution
The facet distribution functionality has been enhanced to provide more accurate results, improving the overall search experience.

### Fixed Bucket-Sort Unwrap Bug
A bug related to the bucket-sort functionality has been resolved, ensuring that sorting operations work as intended.

## Other

### Version Update
This release includes updates from version 0.14.0 to 0.14.1, which may include minor adjustments and optimizations.
