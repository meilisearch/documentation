# v1.25

First released: 2025-11-10

## New Features

### Experimental: Search Personalization
You can now dynamically rerank search results using personalized prompts with Cohere. This feature allows for a more tailored search experience based on user preferences.

### Experimental: Upload Snapshots to S3
Snapshots can now be uploaded directly to S3, enabling efficient streaming and the use of multipart technology to send data chunks in parallel.

## Improvements

### German Word Segmentation Enhancement
The German word segmenter has been improved to skip segmenting unknown words, ensuring that words not in the dictionary are treated as single entities. For example:

```yaml
Source: "Feuchteschutz"
Before: ["Fe", "uc", "ht", "es", "ch", "utz"]
After: ["Feuchteschutz"]
```

### Chinese Text Segmentation Improvement
The segmentation process for Chinese text has been enhanced to prevent the splitting of numbers and English words. For example:

```yaml
Source: "我从2025年开始学习Rust语言。"
Before: ["我", "从", "2", "0", "2", "5", "年", "开始", "学习", "R", "u", "s", "t", "语言", "。"]
After: ["我", "从", "2025", "年", "开始", "学习", "Rust", "语言", "。"]
```

## Other

### Breaking Change: Redaction of Authorization Header
The `Authorization` header value is now redacted in webhooks. This change means that users will no longer receive the key value when retrieving webhooks, enhancing security. Ensure that your UI does not send the `Authorization` header back after it has been redacted.
