# v1.0

First released: 2023-02-06

## New Features

### Improved Language Support
Meilisearch now includes enhanced support for Korean and improved handling of Chinese characters. Key updates include:
- Normalization of Chinese characters into Pinyin.
- Disabling the HMM (Hidden Markov Model) feature in Chinese segmentation.
- Unification of Chinese character variants.

### Enhanced Error Handling
All detailed task responses now include an `error` field. When a task fails, it provides a JSON-formatted error response with clearer `code` and `type` values, improving the user experience.

## Improvements

### Indexing and Search Speed Enhancements
- Memory usage for search requests with multiple long words has been significantly improved.
- Performance of the `exactness` ranking rule for search requests with many words has been enhanced.
- Multi-word synonyms are now translated into phrases, improving result relevancy and stabilizing latency.
- The performance of the `proximity` ranking rule for search requests ending with short words has been improved.

### Migration Simplification
You can now migrate from any old version of Meilisearch (that supports the dump feature) to the latest version using a single dump, making the migration process smoother.

### CLI Improvements
The `download-latest.sh` script has been updated to integrate support for Apple Silicon binaries.

## Other

### Breaking Changes in CLI
- Master keys must now be at least 16 bytes in production environments for security reasons.
- The `--max-index-size` and `--max-task-db` configuration options have been removed as they were not serving their intended purpose.
- The `--log-level` option now only accepts specific arguments: `ERROR`, `WARN`, `INFO`, `DEBUG`, `TRACE`, and `OFF`.

### Changes to Primary Key Inference
Meilisearch will now attempt to infer the primary key from attributes ending with `id`. If multiple candidates are found, you will be prompted to specify the primary key.

### Binary Name Change
The command to install the Meilisearch binary using `apt` has changed to `apt install meilisearch`. For versions prior to v1.0, continue using `apt install meilisearch-http`.
