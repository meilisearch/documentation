# v1.3.5 🦁

Released: 2023-09-20

## Bug fixes

* Revert the fix on the filters about escaping sequences by @Kerollmops in https://github.com/meilisearch/meilisearch/pull/4069

⚠️ This only purpose of this release is to revert a bug fix we made in [v1.3.3](https://github.com/meilisearch/meilisearch/releases/tag/v1.3.3). Indeed #4038 fixes a bug preventing the use `\` character at the end of a value in the `filter` search parameter. Unfortunately, this bug fix introduced a breaking in the `filter` usage: forcing to escape each `\`. We shouldn't have released this kind of bug fix in a patch version.
**If you are already using v1.3.3 or v1.3.4 and your `filter` works without any issue, not need to upgrade to v1.3.5.**
