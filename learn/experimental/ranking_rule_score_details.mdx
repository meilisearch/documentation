# Detailed ranking scores

The `--experimental-score-details` flag allows you to use the `showRankingScoreDetails` search parameter. It adds a `_rankingScoreDetails` object to each document. This object shows details for each ranking rule.

[Visit the GitHub discussion to learn how to use this feature](https://github.com/meilisearch/product/discussions/674)).

Are you using this feature? Meilisearch wants to hear from you! Tell us about your experience in the [GitHub discussion](https://github.com/meilisearch/product/discussions/674). All feedback is useful and helps make Meilisearch better and easier to use.

<Capsule intent="warning">
`showRankingScoreDetails` is an experimental feature. Experimental features are unstable: their API might significantly change and become incompatible between releases. Meilisearch does not recommend using experimental features in a production environment.

Meilisearch makes experimental features available expecting they will become stable in a future release. However, it is not possible to guarantee when and if this will happen.
</Capsule>
