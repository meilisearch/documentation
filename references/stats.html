# Stats

Stats gives extended information and metrics about indexes and the MeiliSearch database.

# Get stat of an index

GET 
/indexes/:index_uid/stats

Get stats of an index.

# Path Variables

Variable Description
index_uid The index UID

# Example

$ curl \
  -X GET 'http://localhost:7700/indexes/movies/stats'

client.getIndex('movies').getStats()

client.get_index('movies').get_stats()

$client->getIndex('movies')->stats();

client.index('movies').stats

client.Stats().Get("movies")

# Response: 200 Ok

{
  "numberOfDocuments": 19654,
  "isIndexing": false,
  "fieldsDistribution": {
    "poster": 19654,
    "release_date": 19654,
    "title": 19654,
    "id": 19654,
    "overview": 19654
  }
}

# Get stats of all indexes

GET 
/stats

Get stats of all indexes.

# Example

$ curl \
  -X GET 'http://localhost:7700/stats'

client.stats()

client.get_all_stats()

$client->stats();

client.stats

client.Stats().GetAll()
