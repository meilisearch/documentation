# Stats

# Get stat of an index

GET
/indexes/:uid/stats

Get stats of an index.

# Path Variables

Variable Description
uid The index UID

# Example

$ curl \
  -X GET 'http://localhost:7700/indexes/12345678/stats'

# Response: 200 Ok

{
  "numberOfDocuments": 19654,
  "isIndexing": false,
  "fieldsFrequency": {
    "poster": 19654,
    "release_date": 19654,
    "title": 19654,
    "id": 19654,
    "overview": 19654
  }
}
